About

F* is a programming language with support for dependent types, which lets you write machine-verified correctness proofs for your code. It uses the Z3 SMT solver to automatically fill in the gaps in the proofs you write. I believe that the features of F* are useful for general purpose programming, and this site documents the things I learn when developing applications in F*.
